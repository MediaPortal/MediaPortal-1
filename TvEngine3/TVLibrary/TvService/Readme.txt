Howto debug TV Service in Visual Studio.
-----------------------

1. Make sure TV sevice is not installed at the computer. Very important.
2. Execute 'Copy Filters To MP Debug & Register.bat'
3. Start Visual Studio
4. Select TV service and set it as startup project.
5. Make sure 'Command line arguments' is set to: '/DEBUG' in the properties tab of the TV service.
6. Press F5 to start debugging.
